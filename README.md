#E-Commerce-Management-System

This is a responsive E-Commerce Website developed using HTML5, CSS3, and JavaScript. The project provides a user-friendly shopping interface and an Admin Panel for managing products. It demonstrates front-end web development concepts, including responsive design, DOM manipulation, form validation, and CRUD operations.
Features
User Features
Browse available products.
View product details.
Responsive design for desktop, tablet, and mobile devices.
About Us page with company information.
Contact Us page for customer inquiries.
Easy navigation and modern user interface.

Admin Features
Add new products.
Update existing product information.
Delete products from the catalog.
Manage product listings through an Admin Dashboard.

Technologies Used
HTML5
CSS3
JavaScript

Installation
1)Clone the repository:
git clone https://github.com/your-username/ecommerce-website.git
2)Open the project folder.
3)In Terminal Start the Local Db Using
npx json-server --watch db.json
4)Run the application by opening index.html in your browser.
